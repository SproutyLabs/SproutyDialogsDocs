Hi everyone! 🌱

Some critical bugs fixed, now you can connect nodes between different dialog trees, and the `Commad/Ctrl-S` shortcut was added to save files and variables! 💾

<!-- truncate -->

## What's Changed

---

- Add support for the StringName type for use in Call Method node parameters by @Snowpire in [#28](https://github.com/SproutyLabs/SproutyDialogs/pull/28)
- Fix several issues in Variable Manager by @Snowpire in [#29](https://github.com/SproutyLabs/SproutyDialogs/pull/29)
- Fix recursion error in node connections that crashed the editor by @Snowpire in [#30](https://github.com/SproutyLabs/SproutyDialogs/pull/30)
- Fix start ID listing error from DialogPlayer in [#31](https://github.com/SproutyLabs/SproutyDialogs/pull/31), reported by @p4tricki in [#27](https://github.com/SproutyLabs/SproutyDialogs/issues/27)
- Add Command/Ctrl-S shortcut to save in [#32](https://github.com/SproutyLabs/SproutyDialogs/pull/32), suggested by @kakoooon in [#26](https://github.com/SproutyLabs/SproutyDialogs/issues/26)
- Remove expression error when returns nothing in [#33](https://github.com/SproutyLabs/SproutyDialogs/pull/33)
- Fix connections between different dialog trees in [#34](https://github.com/SproutyLabs/SproutyDialogs/pull/34)
